# Ночи длинных ковшей

[Интерактивная карта](http://dev.nim579.ru/long-scoops/) объектов самостроя, снесенные в "Ночи длинных ковшей".

Адреса могут быть не точными, т.к. объекты оперативно удаляют с карты. Данные при желании можно редактировать руками.

## Данные

В папке `data` лежат списки адресов. Файлы с постфиксом *_ready* — преобразованные данные с найденными координатами.

Подготовка данных проводится скриптом `./parse.coffee`. В нем адреса для исходных данных прописаны вручную в самом низу (надо будет сделать как-то нормально).
